# Medication App

## 背景
介護現場における服薬管理は属人化しやすく、
記録漏れや飲み忘れ確認のヒューマンエラーが発生しやすい課題があります。

## 目的
服薬状況の可視化と、将来的なデータ蓄積を想定した設計により、
業務効率化と安全性向上を目指します。

## 主な機能
- 朝昼夕の服薬記録
- 服薬状態の確認表示
- パッケージ分割による責務分離設計

## 技術構成
- Java
- パッケージ設計（app / domain / service / ui）
